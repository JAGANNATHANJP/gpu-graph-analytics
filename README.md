# GPU Graph Analytics Framework

A GPU-Accelerated Graph Analytics Framework implementing graph algorithms such as:

- Breadth-First Search (BFS)
- PageRank
- Single Source Shortest Path (SSSP)

## Project Goals

- Build CPU baselines for graph analytics algorithms
- Port algorithms to NVIDIA GPUs using CUDA
- Analyze performance using NVIDIA Nsight tools
- Optimize memory access patterns and parallel execution
- Benchmark CPU and GPU implementations

## Technologies

- C++
- CUDA
- Python
- NVIDIA Nsight Systems
- NVIDIA Nsight Compute
